composerインストール後

php.ini編集（一番最後とsqlite）

下記コマンドを実行
composer global require laravel/installer

下記パスを環境変数に追加
C:\Users\利用者名\AppData\Roaming\Composer\vendor\bin

プロジェクトを作成するディレクトリに移動後以下のコマンドを実行（プロジェクト名は適宜変更）
Laravel new shogiquizzes

composer install

プロジェクト内にcdで移動後
php artisan serve

urlに移動して動作確認


ここからフロントの設定


composer require laravel/ui

php artisan ui vue --auth

node.jsインストール

npm install bootstrap-vue bootstrap

import BootstrapVue from 'bootstrap-vue';
Vue.use(BootstrapVue);
↑メインのjsファイルに追加

npm install

npm install vue-router

npm install vuex

npm run watch
